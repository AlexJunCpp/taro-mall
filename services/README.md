### 二、package命名说明
- form包：输入对象
- model包：输出对象
- entity包：数据库对象
- person包：用户接口
- admin包：管理接口
- inner包：内部接口
- service包：服务类
- repository包：实体对象操作类
- type包：type枚举类型
- status包：状态枚举类型
